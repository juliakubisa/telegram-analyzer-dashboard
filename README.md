### MAIN GOAL 
The main goal of this project was to create an interactive app that will offer an insight into conversations using popular messaging service Telegram. This is an extended version of https://github.com/juliakubisa/telegram-chat-analysis, with basic statistics and plots  not only available to look at, but also displayed in the dashboard made using Dash. 

### KEY FEATURES
- upload your own JSON file with Telegram conversation using "drag and drop" field 
- full-depth conversation analysis in a data range of your choice using customized calendar 
- basic statitics: total and average messages, average words, most active day 
- multiple plots such as: 
  - timeline of your conversation
  - hourly/weekly/monthly messages distribution
  - most popular emoijs
  - n-grams
